# 🎓 ЕГЭ Мастер-тренажёр 2026: База Ошибок

Это адаптивное веб-приложение (SPA) для интерактивной работы над ошибками при подготовке к ЕГЭ по русскому языку. В отличие от стандартных тестов с фиксированными вопросами, это приложение подстраивается под конкретного ученика, позволяя формировать индивидуальную базу сложных слов прямо в процессе обучения.

---

## 🚀 Основные фичи

* **🧠 Принцип «Работа над ошибками»:** База изначально пуста. Вы сами добавляете слова, в которых ошиблись при прорешивании сторонних вариантов, и гоняете их до полного автоматизма.
* **🎯 Система «Мастерство» (0/2):** Чтобы слово считалось выученным и пропало из пула тестирования, на него нужно ответить верно **два раза подряд**. Любая ошибка сбрасывает счётчик мастерства для этого слова в ноль.
* **📱 Полная автономность и кроссплатформенность:** Приложение написано на чистом HTML/JS без внешних зависимостей. Отлично адаптировано под экраны смартфонов и работает без интернета.
* **💾 Умный Экспорт/Импорт (Синхронизация):** Возможность легко переносить свою базу ошибок и прогресс между ПК и мобильным телефоном через один компактный JSON-файл.
* **🎨 Стилизация и кастомизация:** Поддержка 5 контрастных тем оформления (включая тёмные и неоновые) с сохранением выбора пользователя.
* **⚡ Полуавтоматический ввод:** При добавлении слова с пропуском через нижнее подчёркивание (например, `Заг_реть`), интерфейс сам генерирует варианты ответов с буквами **А** и **О**, экономя время на ввод.

---

## 🛠️ Архитектура данных (Локальное хранилище)

Приложение полностью работает в песочнице браузера и использует `localStorage` для персистентного хранения. Данные разбиты на два ключевых объекта:

1. `ege_error_db` — Хранилище карточек, изолированное по идентификаторам заданий (`N4`, `N5`, `N9` и т.д.):

```json
{
  "N9": [
    {
      "id": "err_1716225720000",
      "display": "Вым_кнуть (под дождем)",
      "buttons": ["вымакнуть", "вымокнуть"],
      "correct": "вымокнуть",
      "tip": "Значение 'пропускать жидкость' — пишем О (-МОК-)."
    }
  ]
}

```

2. `ege_mastery` — Карта прогресса изученности слов, связывающая уникальный `id` карточки с текущим уровнем владения (от `0` до `2`).

## 🔄 Инструкция по синхронизации (ПК ↔ Телефон)

Поскольку проект не использует внешние сервера баз данных ради конфиденциальности и автономности, синхронизация между устройствами происходит в три простых шага:

1. **На компьютере:** После завершения сессии подготовки нажмите кнопку **«📥 Экспорт базы»** в главном меню. Браузер скачает файл формата `ege_errors_backup_ГГГГ-ММ-ДД.json`.
2. **Перенос:** Отправьте этот файл себе в Telegram (в «Избранное»), во ВКонтакте или закиньте на любое облако, после чего скачайте его на смартфон.
3. **На телефоне:** Откройте тренажёр на телефоне, нажмите **«📤 Импорт базы»** и выберите скачанный JSON-файл. Приложение предложит два режима:
* **Объединить:** Добавит новые слова с сохранением старых записей на телефоне (рекомендуется).
* **Заменить:** Полностью перезапишет базу телефона данными с ПК.

## 📝 Как запустить проект

1. Скопируйте код приложения в файл с именем `index.html` или перейдите по ссылке: https://h9ekon.github.io/ege-H9ekoN.github.io/.
2. Кликните по файлу дважды, чтобы открыть его в любом современном браузере (Chrome, Яндекс.Браузер, Safari, Firefox).
3. Чтобы использовать тренажёр на телефоне, просто перенесите этот файл `index.html` во внутреннюю память смартфона и откройте его через любой локальный HTML-вьювер или браузер.
